# E-Commerce-Data-Analysis-project
Data-driven analysis of E-commerce sales to identify top products, key customers, and revenue trends using Python and Power BI.


##  Overview
This project focuses on analyzing an **E-commerce sales dataset** to uncover valuable insights about customers, products, and revenue performance.  
Using **Python for data processing** and **Power BI for visualization**, the project demonstrates how data analytics can drive business decisions, optimize product performance, and reduce revenue concentration risks.

---

##  Objectives
- Analyze customer purchasing behavior and product performance.  
- Identify **top-selling products**, **high-revenue segments**, and **sales trends**.  
- Build a **Power BI dashboard** for quick, interactive business insights.  
- Support better decision-making using data-driven storytelling.

---

##  Project Files
| File Name | Description |
|------------|-------------|
| `data.csv` | Cleaned E-commerce transaction dataset used for analysis. |
| `e-commerce_python_script.ipynb` | Jupyter Notebook performing data cleaning, aggregation, and visualization. |
| `Ecommerce_Analysis_Dashboard.pbix` | Power BI dashboard showcasing KPIs and business insights. |

---

##  Tech Stack
- **Languages:** Python  
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn  
- **Tools:** Power BI, Jupyter Notebook  
- **Techniques:** Data Cleaning, Sales Analysis, KPI Tracking, Dashboarding  

---

##  Dashboard Insights
The Power BI dashboard provides a comprehensive view of:
- **Top Customers & Regions:** Revenue contribution and purchasing patterns.  
- **Top-Selling Products:** Units sold, revenue trends, and category performance.  
- **Monthly Sales Analysis:** Seasonal revenue fluctuations and growth rate.  
- **Customer Segmentation:** Identifying loyal and high-value customers.  

---

##  Key Achievements
- Analyzed **5,000+ E-commerce transactions** for actionable insights.  
- Identified **high-revenue products** and **customer segments** contributing to sales concentration.  
- Developed a **Power BI dashboard** for real-time business monitoring.  
- Highlighted **revenue concentration risks** and **improved decision visibility**.
